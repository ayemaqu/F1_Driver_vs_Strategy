# F1_Driver_vs_Strategy
An end-to-end data analytics project analyzing Formula 1 race data from 2010 to 2024 to determine whether race outcomes are driven more by driver skill or team strategy. Built using Python, pandas, SQL, and Tableau

## Research Question
Does winning in F1 come down to the driver, the team strategy, or both? And has that balance shifted depending on the era?

### Data Source
- Formula 1 World Championship dataset by vopani on Kaggle. Download the CSV files and place them in data/raw/.
- Link: https://www.kaggle.com/datasets/rohanrao/formula-1-world-championship-1950-2020

### Project Structure
1. data/raw: Raw CSV files (not tracked by GitHub)
2. data/clean: Cleaned and processed data
3. notebooks: Jupyter notebooks by week
4. src: Reusable Python scripts
5. visualizations: Exported charts and Tableau screenshots

#### Tools Used
- Python, pandas, SQL, Tableau
